# Analysis Method

Use this page to define how the ToolBook should turn watched data into accepted knowledge.

Required behavior:

- Prefer current primary sources and durable datasets.
- Separate domain evidence from operational logs.
- Record what changed, what did not change, and what remains uncertain.
- Promote only accepted analysis into `wiki/reports/latest-analysis.md`.
- Notify only when the report, data, or tool meaningfully changes.
