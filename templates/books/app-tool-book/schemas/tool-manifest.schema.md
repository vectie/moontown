# Tool Manifest Schema

`tool-manifest.json` uses `moontown.app_tool_book.manifest.v1`.

Required fields:

- `schema`
- `book_id`
- `title`
- `purpose`
- `civic_building_id`
- `report_path`
- `app_entrypoint`
- `generated_tool_path`
- `data_path`
- `tool_spec_path`
- `build_command`
- `review_status`
- `last_successful_build_ms`
